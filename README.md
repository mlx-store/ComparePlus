# Compare Product Plus for magento

Compare Product Plus support ajax, quick view compare. Also, you can allow/disallow guest usage, etc...

## Feature
- You can enable/disable extension easy.
- You can use ajax for add, remove, show compare.
- Support fancybox popup.
- Easy add compare url to top link.
- Support disable guest usage.
- Flexible settings for advanced usage.
- W3C XHTML 1.0 Transitional. W3C CSS Valid.
- Fully compatible Chrome, IE7+, Firefox 3+, Flock 0.7+, Netscape, Safari, Opera 9.5
